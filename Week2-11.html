More infomation about coding links,list and images since those are used the most when it comes to starting to develop a responsive webpage

Week 4

Week Four was more focused on CSS. That stands for "Cascading Style Sheets". CSS provides an easy way to control the layout and overrall presentation of a webppage. Css seperated the content from the design. During week four i learned of the three way to add CSS to a web page.

  1. external style sheet
  2. embedded styles
  3. inline styles

Week 5

Week 5 was a bit tougher we started to get more invested into CSS. By this i mean we learned to use the CSS box model for page layout such as spacing, background, and borders. These basic properties are used to control the spacing between elements.

Week 6 and 7

Weeks 6 and 7 touched base on the componets of a Responsive Web Design. Responsive Web Design is important because it gives a webpage the ability to adjust to smaller screens. This includes tablets and other moble devices. I learned that fluid layout is used to adjust the width of a webpage. Also media queries adjusts the appearance of the webpage based on the device. Although i don't quiet understand media queries to well it makes sense on why it is used. The last component to Responsive Web Design is including scalabe images. I made sure to take note of this becuase no one wants a webpage where the images don't scale down to size to adjust the device being used.
